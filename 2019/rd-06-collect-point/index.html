活動辦法

  1. 集點兌換流程：A：累積點數->換折價券->結帳使用。B：累積點數->兌換商品->等待出貨。
  2. 集點兌換說明 : 於東森購物網路通路消費累積發票金額滿500元，即可獲得1點(部份館別不參與此活動)，滿1,000元，獲得2點，依此類推。累積之點數將依照不同商品的兌換條件，可選擇領指定商品之折價券，或是兌換贈品。兌換時間將在集點時間結束後，無取消/退貨者，點數方可兌換折價券或贈品。
  3. 加價購流程 : 於此活動頁面內『集點加價購專區』，點選欲加價購之商品的【領取換購去】按鈕，先領取此商品之專屬折價券(確認領取完後，原累積點數狀態也將改為已使用)，再將此商品加入購物車後，點選【使用折價券】完成結帳。
  4. 兌換贈品流程 : 於此活動頁內『兌換贈品區』，點選欲兌換之商品的【立即免費兌換】按鈕，確認兌換完成後(原累積點數狀態也將改為已使用)，請耐心等待贈品出貨。
  5. 集點時間：2019/7/1~2019/7/31，敬請把握。
  6. 兌換時間：2019/8/16~2019/8/31，逾期失效。
  7. 出貨時間：2019/9/1起陸續出貨，請耐心等待。
  8. 不適用館別：全站前台分類，3C館(包含Apple/手機/平板/穿戴、筆電/桌機/零組件、螢幕/資訊週邊/辦公OA、數位/電玩/電競)、家電館(家電/電視/吸塵器)、旅遊館、票券、電視購物(3C/家電)、品牌旗艦(3C/家電)等專案商品。

注意事項

  1. 兌換資格於兌換折價券/贈品完後，點數狀態將顯示為『已使用』。
  2. 兌換之折價券、加購的商品及兌換贈品數量皆有限，兌換完畢即截止。
  3. 換購商品依訂單順序進行出貨；本次活動為限量商品，數量有限換完為止。
  4. 換購之商品辦理退貨時，必須是全新狀態且包裝完整，否則將會影響退貨權限；辦理退貨完成後，換購點數最晚將於次日中午 12:00 前歸還，返還後之點數使用截止時間與贈給時使用期限相同。
