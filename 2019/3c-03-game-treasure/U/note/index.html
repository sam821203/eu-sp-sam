注意事項

  1. 折扣金1點可折抵現金1元，最高折抵上限為單一商品消費金額之30%。且單筆上限金額為3,000元折扣金，單日最高２萬的抵用限制。
  2. 不回饋也無法使用折扣金之商品類別：加價購、票券類（包含餐券／服務票券／禮券與提貨券等預付儲值商品）、apple系列商品、旅遊、保險及汽機車…等專案商品，無法使用折扣金折抵。
  3. 折扣金不得兌換現金、找零或兌換其他贈品或折抵物流費、安裝處理費等費用。
  4. 網站商品是否適用折扣金/折價券，依結帳頁面(購物車)顯示為準。
  5. 森森購物網所發送之折扣金不可與折價券或紅利金合併使用，僅限於森森購物網網站購買商品時選擇使用，且不可轉讓。
  6. 專人下單、購買非APP通路商品、購買預付型商品及不列入活動累計；使用有效期之禮物卡折抵之金額不列入活動累計。
  7. 折扣金於有效期限內皆可累積抵用；逾期自動無效。
  8. 凡商品置入購物車，未出現優惠折價券欄位或輸入折價券露出「不適用」，即代表此為專案商品，無法參加優惠折價活動。
