  • 【天天抽】於活動期間內，掃描瓶蓋/集合包內QR code並登錄個人資料，即立即開獎。
  • 【加碼抽】於2020/06/03(三)抽出，抽獎全程將由律師見證抽出。
  • 【天天抽】與【加碼抽】皆於2020/06/05(五)於活動網站公告中獎名單。
  • 【加碼抽】若遇天氣等因素停止上班上課，抽獎與中獎名單公告時間則順延，確切時程依主辦單位公告日期為準。
  • 【天天抽】若遇天氣等因素停止上班上課，中獎名單公告時間則順延，確切時程依主辦單位公告日期為準。

兌獎方式
  1. 【天天抽】中獎者: 掃描瓶蓋/集合包內QR code，若為中獎者即可立即兌獎，黑松公司亦將同時以中獎者參加抽獎時提供之個人資訊以簡訊發送中獎通知。
  2. 【加碼抽】中獎者：黑松公司於活動網站上公佈中獎名單後，將依中獎者參加抽獎時提供之個人資訊以e-mail寄送中獎通知函予中獎者。中獎者應依中獎通知函內之指定時間寄回函中所示應備文件及實體「中獎QR code活動瓶蓋/集合包」正本至「台北市大安區信義路四段296號3樓，黑松沙士【沙士70 夢想成金】活動小組收」， 收件截止日以郵戳為憑，逾時即喪失資格。中獎獎品價值超過新台幣20,010元(含)以上者，須依法繳納10%機會中獎所得稅稅金，主辦單位確認以上兌獎資格無誤後，將以匯款或親領方式進行兌獎。同時金額會扣除代扣稅額，並於2020/06/30(二)前兌換完畢。
  3. 活動憑證：【加碼抽】中獎者應保留本次活動中獎瓶蓋/集合包及中獎通知函所示應備文件供黑松公司驗證，活動結束後將不予退還，若中獎活動瓶蓋/集合包與系統登錄不符，或任何不完整、無法辨識、毀損、偽造、經改動的、複製中獎活動瓶蓋/集合包均視為無效，主辦單位有權取消相關兌獎資格，並有權拒絕核發中獎獎品。
  4. 中獎活動瓶蓋/集合包遺失恕無法領獎，將視同放棄中獎權益，不得向主辦單位要求任何補償。
