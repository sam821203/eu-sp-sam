    5%東森幣計算邏輯=依發票實付金額回饋5%，共計最高500枚。(活動區間每位會員最高上限500枚東森幣)

    原商品回饋=配送確認日+15天後匯入；加碼5%東森幣 = 2020/6/8 18:00匯入。
    東森幣使用效期：
    原商品回饋東森幣=匯入日+365天；加碼10倍東森幣 =2020/7/8 23:59，逾期失效。

    1. 小明於APP活動頁面搶到1元折價券，並於日用館消費1,000元商品，使用1元折價券，訂單無取消/退貨，於6/8再送49枚東森幣(依發票實付金額)。(發票實付金額999x0.05= 49枚東森幣)。
    2. 小明於APP活動頁面分別於5/5、5/7皆搶到1元折價券共兩張，並訂購床組、沙發各別皆有使用1元折價券，床組發票實付金額$9,000、沙發發票實付金額$5,000，訂單無取消/退貨，故於6/8再送500枚東森幣。(因活動區間每位會員東森幣回饋5%上限500枚東森幣)
