活動日期

9/1上午11:00 - 9/28下午15:00

流程說明

(需為 2020/9/1 起 加入決勝21點的新玩家)

活動辦法

  1. 活動期間內，邀請好友至遊戲『決勝21點』綁定您的推薦碼 ( 邀請的好友需為2020/9/1起加入『決勝21點』的新玩家 )，確認成功綁定後您與您推薦的好友即可各獲得一張「抽獎卷」，「抽獎卷」可用於當週抽獎活動。
  2. 每週一下午15:00將於『決勝21點』的「得獎名單」中公告當週中獎名單
  3. 獲得「抽獎卷」將用於當週抽獎活動。
  4. 活動期間內，輸入您推薦碼的新玩家若有於『決勝21點』累積下注10枚東森幣，您即可再獲得一張「抽獎卷」。
  5. 活動期間內，新玩家每週於『決勝21點』完成累積下注10枚東森幣，可獲得一張「抽獎卷」。
  6. 抽獎活動日期與獎項內容如下
    抽獎日期 中獎名額 獎項 獎項市價
    9/7 6名 英倫香氛咖啡因洗髮精 750g $199
    9/14 6名 蒲公英雙層衛生紙150抽(一串/28包) $279
    9/21 6名 東森柔嫩抗菌皂草本配方100g*6入 $99
    9/28 6名 耐斯防蟎抗菌微米科技洗衣精暖陽馨 $199

活動資格

  1. 全體東森會員
  2. 於9/1上午11:00 至 9/28下午15:00 邀請新玩家於『決勝21點』輸入您的推薦碼
  3. 綁定您推薦碼的玩家需為2020/9/1起加入『決勝21點』的新玩家，即於2020/9/1前未曾登入過遊戲『決勝21點』的新玩家。
  4. 每位新玩家僅能綁定一組朋友的推薦碼

範例說明

  1. 阿明分享推薦碼給小花，小花於9/10成功綁定小明推薦碼，推薦人阿明可獲得一張「抽獎卷」，綁定推薦碼的新玩家小花也可獲得一張「抽獎卷」。
  2. 綁定阿明推薦碼的新玩家小花已累積下注10枚東森幣，阿明可獲得一張「抽獎卷」(總共累積2張)，小花也可再獲得一張抽獎卷(總共累積2張)。
  3. 新玩家小花於隔週9/15累積下注10枚東森幣，由於新玩家每週累積下注10枚可再獲得一張抽獎卷，故小花可再額外獲得一張「抽獎卷」。
  4. 阿明分享推薦碼給小草，小草於9/29綁定小明推薦碼，由於在非活動期間內綁定，故無法獲得「抽獎卷」。
  5. 小花已於9/10成功綁定阿明的推薦碼，小花欲再綁定阿樹的推薦碼，由於每位新玩家僅能綁定一組朋友的推薦碼，故小花無法綁定阿樹的推薦碼。
  6. 阿明分享推薦碼給東東，東東是2020/8/31加入『決勝21點』的玩家，由於東東非為2020/9/1起加入的新玩家，所以東東無法綁定阿明推薦碼。
  7. 新玩家小花於9/10成功綁定阿明的推薦碼，阿明可獲得一張「抽獎卷」，小花也可獲得一張「抽獎卷」，另一位新玩家西西於9/12成功綁定阿明的推薦碼，阿明可獲得一個「抽獎卷」(累積獲得2張)，西西也可獲得一個「抽獎卷」。

注意事項

  1. 於東森購物App註冊/登入成為會員後，於〔打卡/遊戲〕無須使用東森幣，即可進行推薦碼分享/綁定。
  2. 兌獎流程於活動結束後，本公司將於8/2起七日內，由專人主動與得獎人聯繫。依稅法規定，若得獎人於東森購物兌換贈（獎）品金額達新台幣（下同）1,001元以上未 達20,010元，將於次年初開立免扣繳憑單予得獎人並向主管稽徵機關申報；如達20,010元以上者，將於得獎人扣繳獎品價值10%(外國人20%)之機會中獎所得稅後，且出具繳款收款及中獎通知函後，東森購物始交付獎品並將於次年開立扣繳憑單予得獎人。
  3. 獎品配送區域限臺灣本島地區（外、離島地區請利用臺灣親友地址作為收貨地址），部分偏遠地區需加收額外運費（區域及實際運費由東森購物判定）。若因會員資料填寫不完整者，或活動後變更資料（如姓名、手機、地址等）導致無法成功配送者，嗣後恕不補發。
