  • 注意事項
    1. 收到商品後，若您還不確定是否將會辦理退貨，請勿將產品連上網路，避免開始保固時間。
    2. 未確定保留商品前，勿拆用隨和附贈配件，以免影響您的退貨權益。
    3. 10日猶豫期並非試用期，恕不提供試用、體驗等服務。如於猶豫期內申請退貨，需將商品恢復原狀。(含包裝、配件等；產品一經註冊可能需要酌收額外的費用)
    4. 主商品/配件不可體驗，不可拆封。收到商品後，務必先仔細確認商品正確、外觀可接受，確定要購買後再開機使用。
    5. 退/換貨時請用原紙箱包裝，並保留原配送單。※是否可單一品項退貨?是否有限定商品?
  • 常見問題
    • Q：請問是否能透過其他導購平台連結(如:Line、ShopBack)仍享有優惠價？
    A：活動相關規範以東森購物網公告為主。
    Q：如有其他商品採購需求，是否能成立專案訂單？
    A：請填寫大宗採購專案表單，留下您的需求及希望付款方式，後續將由專員與您聯絡。
    Q：此專案是否適用其他類別的商品(家電、生活用品...等等)？
    A：此專案目前僅開放通訊類別商品(手機、平板、穿戴裝置、通訊周邊)，敬請見諒。若您有其他商品的採購需求，也可填寫表單，我們會轉至相關單位人員進行回覆。
    Q：大宗企業採購之訂單金額含運費嗎？
    A：商品報價皆為含運價格。
    Q：為何大宗採購專案的報價比直接在賣場下單還貴？
    A：因不定時會有「限時」、「限量」的折扣價，此優惠價恕不適用大宗採購專案。
